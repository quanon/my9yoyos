# CLAUDE.md

Claude Code がこのリポジトリで作業する際のガイダンスを提供するファイルです。

## プロジェクト概要

「私を構成する 9 つのヨーヨー」

自分を構成する9つのヨーヨーを選んでシェアできる Web アプリ。
お気に入りのヨーヨーを 3x3 グリッドに並べ、URL でシェアできる。

## 技術スタック

- **フレームワーク**: React + TypeScript (Vite)
- **UIライブラリ**: DaisyUI + Tailwind CSS
- **パッケージマネージャー**: npm
- **ルーティング**: React Router v6
- **状態管理**: React state / URLパラメータ (シェア用)

## 主な機能

1. **3x3 ヨーヨーグリッド** — 最大 9 つのヨーヨーを選択、番号バッジ付きで表示
2. **検索・選択 UI** — ヨーヨー名・ブランドで検索してグリッドに追加
3. **URL シェア** — 選択内容をURLエンコードしてシェア (読み取り専用ビュー)
4. **読み取り専用シェアビュー** — 制作者名の表示と「空の状態から作り直す」ボタン

## ディレクトリ構成

```txt
src/
  components/
    YoyoGrid.tsx        # 3x3 グリッド表示
    YoyoCard.tsx        # 番号バッジ付き個別カード
    YoyoSearch.tsx      # 検索・選択 UI
    ShareBanner.tsx     # 読み取り専用時の「共有リンク閲覧中」バナー
  pages/
    HomePage.tsx        # メイン編集ページ
    SharePage.tsx       # 読み取り専用シェアビュー
  data/
    yoyos.ts            # ヨーヨーデータ（名前・ブランド・画像 URL など）
  utils/
    share.ts            # シェア用 URL エンコード/デコード
  App.tsx
  main.tsx
```

## UI/UXガイドライン

- 参考デザインに準拠：明るい背景、カードベースのグリッド、番号バッジ
- DaisyUIコンポーネントを積極活用（btn, card, badge, input, modal, alert）
- モバイルファーストレイアウト（最大幅430px程度、スマホアプリ風に中央寄せ）
- シェアバナー：読み取り専用時はオレンジ/アンバーのアウトライン枠で上部に表示
- グリッド：3列、正方形カード、カバー画像＋左上に番号オーバーレイ
- 下部カウンター：「N / 9 選択済み」

## コーディング規約

- TypeScript strictモード
- 関数コンポーネント + フックのみ (クラスコンポーネント不使用)
- named export を優先
- コンポーネントは小さく単一責任に
- 不要な抽象化をしない: シンプルに保つ
- コードコメントは英語、UI テキストは日本語

## コマンド

```bash

npm run build    # プロダクションビルド
npm run preview  # プロダクションビルドのプレビュー
npm run lint     # リント実行
```
